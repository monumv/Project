Fetch All Lead Lists

Use this API to retrieve the entire collection of existing lead lists. You can use this lead list to get all lead and related campaign data.

Getting Help

  • Be sure to set the proper authorization header for your deployment.
  • Click here to get detailed information of all the attributes existing in the page.
  • For broadcast list's leads operation, check its API reference page.

Sample Request

Lead lists can be fetched using the following manner:

curl --request GET \

  • --url https://api-smartflo.tatateleservices.com/v1/broadcast/lists \
  • --header 'accept: application/json' \
  • --header 'authorization: Bearer yourtokenhere'

Response Variables

Variable Name Description Data Type
id The unique ID of the lead list. For example, 7006. Integer
name The name of the lead list. For example, Home loan lead list. String
description The description related to lead list. For example, list for home loan leads. String
field_map The field details associated with the lead list. For example, Name, Address, Email. Array of string
assigned_to The broadcast to which this list is assigned. For example, 2236. String
